Температурные режимы экструзии LDPE, LLDPE, HDPE и PP

Автор: Игорь Кряжев, технолог-консультант по переработке вторичных полимеров

1. Введение: почему температура и ПТР определяют успех экструзии

Температура экструзии напрямую связана с показателем текучести расплава. При настройке температурного профиля для выдува пленки или плоскощелевой экструзии листа нужно учитывать, что каждый полимер имеет свой диапазон ПТР, при котором процесс стабилен. Для вторичных полимеров ПТР может быть выше из-за термодеструкции, поэтому температуры экструзии часто снижают на 10-20°C по сравнению с первичным сырьем. Задача технолога - подобрать такой профиль, чтобы расплав заполнял форму без пульсаций, а готовое изделие не имело дефектов.

2. Краткая таблица рабочих диапазонов ПТР для экструзии

LDPE для выдува пленки - ПТР 1,5-3,5, для листа - 1-3 г/10мин. LLDPE для выдува пленки - ПТР 1,5-3, для листа - 1-2,5 г/10мин. HDPE для выдува пленки - ПТР 0,8-2, для листа - 0,5-1,5 г/10мин. PP для выдува пленки - ПТР 2-5, для листа - 1-3 г/10мин.

3. LDPE

Температура плавления LDPE 105-115°C. При выдуве пленки из LDPE с ПТР 2-3 используют профиль: зона 1 140-150°C, зона 2 150-170°C, зона 3 170-190°C, зона 4 180-190°C, головка 170-190°C. При переработке вторичного LDPE с ПТР 4-7 температуры снижают на 10-20°C. Для плоскощелевой экструзии листа из LDPE профиль смещается вниз: зона 1 130-140°C, зона 2 140-160°C, зона 3 160-180°C, зона 4 170-180°C, плоская головка 160-180°C.

4. LLDPE

LLDPE более вязкий и требует более высоких температур. Температура плавления 120-130°C. Для выдува пленки из LLDPE с ПТР 2-3 профиль: зона 1 160-170°C, зона 2 170-190°C, зона 3 190-210°C, зона 4 200-210°C, головка 190-210°C. При переработке вторичного LLDPE из стрейч-пленки с ПТР 2,5-4,5 профиль используют без изменений. Для плоскощелевой экструзии листа: зона 1 150-170°C, зона 2 170-190°C, зона 3 190-210°C, зона 4 190-200°C, плоская головка 180-200°C.

5. HDPE

HDPE требует самых высоких температур. Температура плавления 125-135°C. Для выдува пленки из HDPE с ПТР 1-2 профиль: зона 1 170-190°C, зона 2 190-210°C, зона 3 210-230°C, зона 4 220-230°C, головка 200-220°C. При переработке вторичного HDPE из канистр с ПТР 0,5-1 температуры поднимают до 230-240°C. Безопасный способ - смешение с 20-30% LLDPE.

6. PP

PP имеет температуру плавления 160-170°C. Для выдува пленки из PP с ПТР 3-5 профиль: зона 1 180-200°C, зона 2 200-220°C, зона 3 220-240°C, зона 4 230-240°C, головка 210-230°C. При переработке вторичного PP из биг-бэгов с ПТР 8-10 температуру снижают до 190-210°C. Плоскощелевая экструзия листа: зона 1 170-190°C, зона 2 190-210°C, зона 3 210-230°C, зона 4 220-230°C, плоская головка 200-220°C.

7. Особенности настройки для вторичных полимеров

При переработке вторичных полимеров ПТР может отличаться. При повышении ПТР на 2 единицы температуру снижают на 15°C. Для смеси вторичного HDPE с ПТР 0,5 и 20% LLDPE температуры экструзии снижаются на 20°C.
