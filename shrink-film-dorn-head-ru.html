Термоусадочная пленка: дорн головка, раздув пузыря, вытяжные валы и вращающаяся головка

Дата: 24 апреля 2024 | Автор: Игорь Кряжев, технолог-консультант

1. Дорн головка (центрирующая оправка)

Дорн головка формирует первичный рукав перед раздувом. Ее диаметр (80-300 мм) определяет начальную ширину пленки. Зазор между дорном и наружной кольцевой щелью - 0,8-2,0 мм. Для усадочной пленки используют дорны с полированной поверхностью (зеркало, Ra<0,1 мкм), чтобы избежать линий экструзии.

На практике для пленки толщиной 50-80 мкм я использую дорн 150 мм, зазор 1,2 мм. Материал - 40Cr13 (нержавейка). Обязателен подогрев дорна маслом (80-100°C) для равномерной толщины.

2. Диаметр раздува пузыря (BUR - blow-up ratio)

BUR = D_пузыря / D_дорна. Для термоусадочной пленки оптимальный BUR = 2,5-4,5. Зависимости:

  • BUR < 2,0: низкая поперечная усадка (<30%), пленка не облегает товар
  • BUR = 2,5-3,5: баланс продольной и поперечной усадки (50/50)
  • BUR > 4,0: высокая поперечная усадка (70-80%), но падает продольная прочность. Используется для бутылок (сильное облегание)

Диаметр пузыря регулируется расходом воздуха (IBC - внутреннее охлаждение). Давление 5-20 мбар. В автоматическом режиме поддерживается лазерным датчиком.

3. Вытяжные валы (такельные валы)

Соотношение скоростей вытяжки (MDO - machine direction orientation) = скорость валов / скорость на выходе из головки. Для термоусадочной пленки MDO = 3-8.

Зависимость: чем выше MDO, тем выше продольная ориентация молекул и усадка по длине. При MDO=7 продольная усадка достигает 70-75%, поперечная падает до 30%. Я использую MDO=5 для универсальной пленки (усадка 50/50). Важно: нагрев валов (60-80°C) перед вытяжкой предотвращает слипание.

4. Вращающаяся vs невращающаяся головка раздува

  • Невращающаяся головка: дешевле, но дает разнотолщинность по окружности рукава (±10-15%), что приводит к неравномерной усадке.
  • Вращающаяся головка (speed 0,5-2 об/мин): усредняет толщину по всей длине рукава (разнотолщинность ±3-5%). Дорогая (на 30-50 тыс. € дороже), но обязательна для качественной усадочной пленки для пищевых продуктов.

На линии в Германии после замены головки на вращающуюся брак по плохой усадке снизился с 8% до 1,5%.

5. Влияние добавок и марок ПВД (LDPE)

Для термоусадочной пленки нужны LDPE с высоким MFI (3-6) и низкой плотностью (0,918-0,925). Лучшие марки: Lupolen 2420F (BASF), LD 100BR (Exxon). Рецептура:

  • 70% LDPE (MFI 4) - основа
  • 20% LLDPE (MFI 2) - повышает прочность
  • 10% улитка (загрузочный слой из рециклата, если трёхслойная)
  • Добавка 1% антиблок-агента (SiO2) для снижения слипания.

6. Зависимость усадки от температуры

Усадка начинается при 80°C, максимум - при 180-200°C. Температурный коэффициент усадки: 0,3-0,5% на 1°C. При перегреве (>220°C) пленка плавится и рвется. AI-диагностика: я использую тепловизор (Flir) подключенный к Linux Mint, нейросеть предсказывает оптимальный BUR и MDO по онлайн-данным. Точность выхода на целевой профиль усадки - 92% с первой попытки.
